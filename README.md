# Team10_TermProject

## Usage
* Use deep learning to generate depth map: 
```
cd DenseDepth
python -B test.py
```
* Draw 3D Bounding Box according to Depth Map: 
```
python -B draw_DenseDepth.py
```

```
python -B draw_stereoBM.py
```
